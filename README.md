# PicFusion
### ***커스텀 갤러리, 이미지 합성 및 편집 Application***
<br><br>
## 01. 프로젝트 개요
-  내 앨범을 커스텀 해줍니다
-  이미지를 합성해 저장 합니다
<br><br>

## 02. 프로젝트 구성
### ***개발 기간***   
2024.12.19 ~ 2024.12.26 (1주)
<br><br>

### ***담당***     
Android: 이동수     
<br><br>

### ***사용 기술***
**Jetpack Compose**       
**MVI, Clean Architecture**      
**Multi Module**        
**StateFlow, SharedFlow**      
**Coroutine**      
**Coil**      
<br><br>
